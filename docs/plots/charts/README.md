# Charts for the brittle star locomotion module

The charts are made with the Vega-lite API. To install Vega-lite, run `npm install` inside the `/plots` directory.

## Trajectory charts

The two trajectory charts `actions_trajectory.js` and `positions_trajectory.js` are automatically generated when our brittle star locomotion module is ran in evaluation mode with `--mode eval`. They can be generated from the command line as such:

```bash
cd plots # Ensure you are in the plots directory
node index.js eval out/eval_actions.csv out/eval_breakpoints.csv out/eval_positions.csv
```

The arguments to `index.js` are the paths to the evaluation trajectories generated by our brittle star locomotion module.

## Learning curve comparison

The `learning_curve_comparison.js` chart does not automatically get generated in evaluation mode. The chart is made to compare two learning curves of two trained brittle star models, based on data generated by `wandb`. It is our recommendation to manually download the csv on the wandb website where you compare two models adn then first clean the csv with the `script/preprocess_wandb_learning_curve_comparison.sh` script, before making the chart with:

```bash
cd plots # Ensure you are in the plots directory
node index.js comparison out/wandb_learning_curve_comparison.csv
```
