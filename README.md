# Software Engineering Lab 3 - Brittle Star Locomotion

## Abstract

Building robots that can deal with the real world's challenges is a difficult task.
In this lab, we explore the concept of enabling robots to deal with damage and continue functioning, inspired by the brittle star's ability to adapt its locomotion when one of its arms is damaged.
In order to achieve this, we will implement a control system based on central pattern generators (CPGs) and use multi-agent reinforcement learning (MARL) to allow the robot to adapt its gait in response to damage.
